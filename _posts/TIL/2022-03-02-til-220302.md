## Github 저장소 Clone

```
$ git clone {저장소 URL}
```



## 특정 Branch Clone

```
$ git clone -b {Branch name} --single-branch {저장소 URL}
```



## Git 저장소 push & pull

### main push

```
$ git add .
```

```
$ git commit -m "커밋내용"
```

```
$ git push
```

### main pull

```
$ git pull
```



## 특정 Branch push & pull

### Branch push

```
$ git add .
```

```
$ git commit -m "커밋내용"
```

```
$ git push origin {Branch name}
```

### Branch pull

```
$ git pull origin {Branch name}
```
