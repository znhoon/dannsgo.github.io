## 숫자

```js
alert(1);
```

- 큰따옴표나 작은따옴표가 붙지 않은 숫자는 숫자로 인식



<br/>



## 덧셈

```js
alert(1+1);
alert(1.2 + 1.3);
```



<br/>



## 곱셈

```js
alert(2 * 5);
```

- `*` (에스터리스크, Asterisk)



<br/>



## 나누기

```js
alert(6 / 2)
```

- `/`  (슬래쉬, slash)



<br/>



## 제곱

```js
Math.pow(3,2);	// 9, 3의 2승
```



<br/>



## 반올림

```js
Math.round(10.6);	// 11, 10.6을 반올림
```



<br/>



## 올림

```js
Math.ceil(10.2);     // 11,  10.2를 올림
```



<br/>



## 내림

```js
Math.floor(10.6);    // 10,  10.6을 내림
```



<br/>



## 제곱근

```js
Math.sqrt(9);        // 3,   3의 제곱근
```



<br/>



## 난수(랜덤한 숫자)

```js
Math.random();       // 0부터 1.0 사이의 랜덤한 숫자
```
