## Variables

```
console.log(); // 콘솔 출력
console.log(123123);
console.log("lalalalala"); // 큰따옴표 && 작은따옴표 모두 가능
```

```
console.log(6 + 2);
console.log(6 * 1);
console.log(6 / 2);
```

```
const a = 5;

console.log(a + 2);
console.log(a * 1);
console.log(a / 2);
```

```
const a = 5;
const b = 2;

console.log(a + b);
console.log(a * b);
console.log(a / b);
```

```
const veryLongVariableName = 0; // js에서 주로 사용하는 variables 작명 == camelCase
very_long_variable_name = 0 // python에서 주로 사용하는 variables 작명 == snake_case
```

```
const a = 5;
const b = 2;
const myName = "홍길동";

console.log(a + b);
console.log(a * b);
console.log(a / b);
console.log("hello " + myName);
```



## const and let

```
const a = 5;
const b = 2;
let myName = "홍길동";

console.log(a + b);
console.log(a * b);
console.log(a / b);
console.log("hello " + myName);

myName = "홍씨";

console.log("your new name is " + myName);
```

- const => 변수명 업데이트 불가능
- let => 변수명 업데이트 가능
- const를 기본적으로 사용
- var 사용 X => 규칙 없음
